GEMERA TOKEN DISTRIBUTION

GEMERA Token Distribution is currently open and will be available until April 15, 2018, or when GEMERAS are sold out. 1,000,000 (one million) ERC-20 compatible GEMERAS will be distributed.

How to Participate?

Send Ethers (ETH) to the GEMERA Token Distribution smart contract address displayed below or use the GEMERA Metamask interface (Metamask).

Option 1

In Google Chrome add the Metamask Wallet Extension and then go to the GEMERA Metamask interface:

  • GEMERA METAMASK INTERFACE

Option 2

Send Ethers (ETH) to the GEMERA Token Distribution address from a compatible wallet:

Compatible Wallets

  • MyEtherWallet
  • Ethereum wallet

DO NOT USE INCOMPATIBLE WALLETS

There are many incompatible wallets, please check to make sure that your wallet meets the compatible wallet criteria before sending any Ether. The following wallets are known to be incompatible wallets; this is not a complete list. Do not use any of the following wallets to participate in the GEMERA Token Distribution:

  • Any Bitcoin exchange
  • Any Ethereum exchange
  • Bitfinex
  • Bittrex
  • Bitstamp
  • Cex.io
  • Coinbase
  • Exodus
  • Jaxx
  • Kraken
  • Poloniex
